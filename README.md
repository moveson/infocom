Infocom
================

Infocom is a text-based adventure engine inspired by its namesake in the 1970s and 1980s.

To play:
1. Clone this repo
2. `cd infocom`
3. `gem install bundler` (if not already installed)
4. `bundle`
5. `ruby infocom.rb`
